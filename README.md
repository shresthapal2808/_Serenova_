# Serenova
A mental wellness web application focused on relaxation, self-expression, and emotional connection.

---

## 📌 Description

Serenova is a mental wellness platform designed to help users relax, express their thoughts, and find inner peace.

⚠️ Note: This application does not provide medical advice, diagnosis, or treatment. It is intended solely for emotional support and relaxation.

## 🛠️ Tech Stack

**Frontend:**
- HTML, CSS, JavaScript
- AI tools integration

**Backend:**
- Django
- Django REST Framework

**Database:**
- SQLite (Development)
- PostgreSQL (Production)

**Deployment:**
- Azure Virtual Machine (Ubuntu)
- Gunicorn (WSGI server)
- Nginx (Reverse proxy server)

## ⚙️ Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/yourusername/serenova.git
cd serenova
```
2. Install dependencies
```bash
pip freeze > requirements.txt
```
3. Apply migrations
```bash
python manage.py migrate
```
4. Run the development server:
```bash
python manage.py runserver
```
